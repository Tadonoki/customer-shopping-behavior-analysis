# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI**. This project demonstrates the complete analytics workflow from data cleaning to business insights and interactive dashboard development.

---

## 📖 Project Overview

The objective of this project is to analyze customer purchasing behavior and generate actionable business insights.

The workflow includes:

- Data Cleaning with Python
- Data Storage in PostgreSQL
- Business Analysis using SQL
- Interactive Dashboard in Power BI
- Business Recommendations

---

## 🎯 Objectives

- Understand customer purchasing behavior.
- Identify high-value customer segments.
- Analyze spending patterns.
- Evaluate subscription performance.
- Compare shipping methods.
- Discover top-performing products.
- Create an interactive business dashboard.

# 📊 Business Questions

The project answers the following business questions:

1. Revenue generated by male vs female customers.
2. Customers using discounts while spending above average.
3. Top 5 highest-rated products.
4. Average purchase comparison by shipping type.
5. Subscriber vs non-subscriber performance.
6. Products with the highest discount dependency.
7. Customer segmentation (New, Returning, Loyal).
8. Top 3 products in each category.
9. Relationship between repeat buyers and subscriptions.
10. Revenue contribution by age group.

# 📌 Key Insights

- Male customers generated higher total revenue than female customers.
- Young Adult customers contributed the highest revenue.
- Loyal customers represented the largest customer segment.
- Express Shipping customers spent slightly more on average.
- Several products rely heavily on discounts to drive sales.
- High-rated products present strong opportunities for marketing campaigns.

---

# 💡 Business Recommendations

- Improve subscription benefits to increase customer retention.
- Strengthen loyalty programs for repeat customers.
- Optimize discount strategies to maintain profitability.
- Promote highly rated and frequently purchased products.
- Develop targeted marketing campaigns for high-value customer segments.

# 🚀 How to Run

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/customer-shopping-behavior-analysis.git
```

### 2. Install Python Libraries

```bash
pip install pandas sqlalchemy psycopg2-binary
```

### 3. Open Jupyter Notebook

```bash
jupyter notebook
```

### 4. Import Dataset

```python
import pandas as pd

df = pd.read_csv("dataset/customer_shopping_behavior.csv")
```

### 5. Load Data into PostgreSQL

Execute the notebook to import the cleaned dataset into PostgreSQL.

### 6. Run SQL Queries

Open:

```
sql/customer_behavior.sql
```

Execute each query in pgAdmin or PostgreSQL.

### 7. Open Dashboard

Open

```
Dashboard/customer_behavior_dashboard.pbix
```

using Microsoft Power BI Desktop.
